## Technical Architecture Summary

**Two-brain system:**
- **Grok (System 2 / Cloud):** High-level reasoning, planning, complex judgment. Fires only when needed.
- **Digital Optimus (System 1 / Local Edge):** Real-time screen watching, clicking, typing, form-filling. Runs on Tesla AI4 chip ($650). Zero cloud latency for routine tasks.

**Why this matters vs. cloud-only agents (Claude Computer Use, OpenAI Operator):**
- Cloud agents require a round-trip screenshot per action → latency + cost per token
- Digital Optimus runs locally → near-zero marginal cost per action, real-time speed

**Scale architecture:**
- Parked Teslas → local AI compute nodes
- Supercharger network (7GW spare capacity) → distributed inference data centers
- Idle Optimus robots → switch from physical to digital tasks overnight
- SpaceX AI satellites → solar-powered, space-cooled inference in orbit

**One unified model across three products:**
- FSD (Full Self-Driving)
- Physical Optimus (humanoid robot)
- Digital Optimus (computer agent)
All improve simultaneously from each other's training data.
